## English

### 1. Login and Basics
- Start the app, choose a language, and log in.
- You can switch between light/dark theme.

### 2. Daily Report
- Select Date/Shift/Area and save the basic info.
- Fill in Key Output / Issues / Countermeasures and save.

### 3. Attendance
- Enter Regular / Contract attendance data and save.

### 4. Equipment / Lot
- Create and edit equipment logs and lot logs.

### 5. Delay List
- Search by date range (Start/End).
- Import Excel/CSV/TXT, then upload after review.
- Double-click a cell to edit directly (Date supports manual input).
- Right-click to delete selected rows.

### 6. Summary Actual
- Search by date range (Start/End).
- Import Excel/CSV/TXT, then upload after review.
- Right-click to delete selected rows.

### 7. Calendar
- Date fields are editable.
- Use the ?Today? button in the calendar popup.
