# 使用者操作手冊

## 1. 登入與語言
1. 啟動程式後先選擇語言。
2. 輸入帳號密碼登入。

## 2. 基本資訊（操作門檻）
1. 在日報表頁面選擇 **日期 / 班別 / 區域**。
2. 按下「儲存」後，其他功能頁面才會解鎖。
3. 右上角會顯示目前選定的日期/班別/區域，作為所有資料的關聯基準。

## 3. 日報表摘要
- 在摘要欄位輸入 Key Output / Issues / Countermeasures。
- 與日報表基本資訊同筆資料保存。

## 4. 出勤記錄
- 正職與契約欄位同頁輸入：定員 / 出勤 / 欠勤 / 理由。
- 儲存後會寫入對應的日報表。

## 5. 設備異常 / 異常批次
- 可新增多筆記錄。
- 會與當前日報表關聯。

## 6. Delay List / Summary Actual
- 流程：**匯入 Excel → 暫存 → 上傳**。
- 未上傳的資料不會寫入 SQLite。
- 可依日期區間查詢與編輯。

## 7. 總結統計
1. 設定統計開始/結束日期。
2. 按下「確定」產生出勤統計表與圖表。
3. 出勤率為折線圖，出勤人數為堆疊柱狀圖。

## 8. 異常歷史查詢
- 支援日期區間 + 班別/區域篩選。
- 分別顯示設備異常與異常批次兩張表。

## 9. 覆寫規則
- 同日期 + 班別 + 區域的日報表與出勤資料會覆寫舊資料。
- 設備異常與批次異常資料維持累加。

## 10. 常用注意事項
- 日期欄位使用日曆選擇，不需手動輸入。
- 若畫面內容過長，可使用滾動條檢視完整內容。
