# 管理者操作指南

## 1. 使用者管理
- 新增/更新/刪除使用者帳號。
- 可重設密碼。
- 管理員帳號刪除前會有確認提示。

## 2. 翻譯資源管理
- 可匯入/匯出 JSON 翻譯資源。
- 建議先匯入，再進行修改，最後匯出保存。
- 語言檔位置：`frontend/public/locales/*.json`

## 3. 班別與區域管理
- 新增/更新/刪除班別與區域。
- 更新後會同步到日報表與篩選選單。

## 4. 系統設定
- 設定 **自動備份** 與 **備份間隔 (天)**。
- 設定值保存於 `handover_settings.json`。
- 若需要實際備份排程，請搭配外部排程或手動備份 `handover_system.db`。

## 5. 版本一致性
- 標題列與左下角版次需一致，版本以 `frontend/public/locales/*.json` 為準。
