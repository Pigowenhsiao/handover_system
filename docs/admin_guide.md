# 管理者ガイド

## 1. ユーザー管理
- ユーザーの追加/更新/削除。
- パスワードのリセットが可能。
- 管理者アカウント削除時は確認ダイアログを表示。

## 2. 翻訳リソース管理
- JSON 翻訳リソースのインポート/エクスポート。
- 推奨フロー：インポート → 修正 → エクスポート。
- 言語ファイル位置：`frontend/public/locales/*.json`

## 3. シフト/エリア管理
- シフト/エリアの追加・更新・削除。
- 更新内容は日報とフィルター選択に反映。

## 4. システム設定
- **自動バックアップ** と **バックアップ間隔（日数）** を設定。
- 設定は `handover_settings.json` に保存。
- 実際のバックアップは外部スケジュールまたは手動バックアップが必要。

## 5. バージョン整合性
- タイトルと左下のバージョン表示を一致させる。
- バージョンは `frontend/public/locales/*.json` を基準とする。
