# 多言語機能の説明

## 概要
本システムは単機デスクトップアプリであり、REST API は提供しません。多言語リソースはローカル JSON ファイルを使用し、LanguageManager が読み込みと切替を行います。

- 対応言語：日本語 (ja)、中文 (zh)、英語 (en)
- 言語ファイル：`frontend/public/locales/*.json`
- 既定言語：日本語

## リソース構造
JSON はネスト構造で管理します。
```json
{
  "header": {
    "title": "電子引き継ぎシステム",
    "languageSwitch": "言語切替"
  },
  "common": {
    "save": "保存",
    "cancel": "キャンセル"
  }
}
```

## 切替フロー
1. ログイン画面または主画面で言語を選択。
2. LanguageManager が即時に文言を更新。
3. キーが不足する場合は既定文言にフォールバック。

## 管理者の運用
- 管理画面から翻訳リソースのインポート/エクスポートが可能。
- 推奨フロー：インポート → 修正 → エクスポート → `frontend/public/locales/*.json` に上書き。

## 注意事項
- 言語選択は単一セッション内のみ保持されます。
- チャートは CJK フォントでフォールバックし、中/日文字化けを回避します。
