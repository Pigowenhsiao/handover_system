# 電子引き継ぎシステム V 0.1.4 - クイックスタート

## 🚀 すぐに起動

```bash
python handover_system.py
```

## 💻 システム特性

- **モダン UI**：サイドバー + コンテンツエリア
- **多言語対応**：日/中/英の即時切替
- **日報フローの門限**：基本情報保存後に他機能を操作可能
- **統計レポート**：出勤統計表 + 折線図 + 積み上げ棒グラフ
- **異常履歴照会**：設備/ロットの二表で検索
- **Delay List / Summary Actual**：Excel インポート + アップロード
- **テーマ切替**：ライト/ダーク即時切替

## 🎯 コア機能

### 📋 日報
- 日付/シフト/エリアの選択
- サマリー欄の入力

### 👥 出勤記録
- 正社員/契約社員を同ページで入力
- 出勤率を自動計算

### ⚙️ 設備異常 / 📦 異常ロット
- 複数記録の追加
- 日報に紐づけて保存

### 📊 出勤統計
- 期間指定
- 出勤率折線図
- 出勤人数の積み上げ棒グラフ

### 🧾 異常履歴
- 期間 + シフト/エリアで絞り込み

### ⚙️ システム管理
- ユーザー管理
- 翻訳資源管理
- シフト/エリア管理

## 🌐 言語切替

右上の言語セレクターから：日本語 / 中文 / English

## 🔒 パスワード要件

- 8 文字以上
- 大文字 + 小文字 + 数字 + 記号

## 🧩 別の起動方法

依存チェックなしでモダン UI を起動する場合：

```bash
python start_modern_ui.py
```
