# 電子交接系統 v2.2 - 快速入門

## 🚀 快速啟動

```bash
python run_modern_system.py
```

## 💻 系統特性

- **現代化 UI**：側邊欄 + 分頁內容
- **多語言支援**：日/中/英即時切換
- **日報流程門檻**：先存基本資訊再操作其他功能
- **統計報表**：出勤統計表 + 折線圖 + 堆疊柱狀圖
- **異常歷史查詢**：設備/批次雙表格篩選
- **Delay List / Summary Actual**：Excel 匯入 + 上傳

## 🎯 核心功能

### 📋 日報表
- 日期/班別/區域選擇
- 摘要欄位輸入

### 👥 出勤記錄
- 正職/契約同頁輸入
- 自動計算出勤率

### ⚙️ 設備異常 / 📦 異常批次
- 多筆記錄新增
- 與日報表關聯

### 📊 總結統計
- 日期區間篩選
- 出勤率折線圖
- 出勤人數堆疊柱狀圖

### 🧾 異常歷史
- 日期區間 + 班別/區域篩選

### ⚙️ 系統管理
- 使用者管理
- 翻譯資源管理
- 班別/區域管理

## 🌐 語言切換

點擊右上角語言選擇器：日本語 / 中文 / English

## 🔒 密碼要求

- 至少 8 個字元
- 大寫字母 + 小寫字母 + 數字 + 特殊字元

## 🧩 其他啟動方式

如需直接啟動現代化介面（不含依賴檢查）：

```bash
python start_modern_ui.py
```
