# 電子交接系統 v2.0 - 快速入門

## 🚀 快速啟動

```bash
cd /home/pigo/Documents/python/handover_system
python run_modern_system.py
```

## 💻 系統特性

- **現代化 UI**: Material Design 風格
- **多語言支援**: 日/中/英即時切換
- **密碼安全**: 強度驗證與加密存儲
- **即時回饋**: 操作狀態顯示
- **統計面板**: 出勤率與異常統計

## 🎯 核心功能

### 📋 日報表
- 填寫每日生產交接資訊
- 記錄 Key Machine Output / Issues / Countermeasures

### 👥 出勤記錄
- 正社員與契約社員分欄輸入
- 即時出勤率計算
- 色彩視覺提示

### ⚙️ 設備異常
- 設備故障記錄
- 圖片上傳
- 處理追蹤

### 📦 異常批次
- 批次異常追蹤
- 處置狀況記錄

### 📊 總結
- 工作總結
- 統計資訊

### ⚙️ 系統管理
- 使用者管理
- 翻譯資源管理
- 密碼重設

## 🌐 語言切換

點擊右上角語言選擇器，切換：日本語 / 中文 / English

## 🔒 密碼要求

強密碼必須包含：
- 至少 8 個字元
- 大寫字母
- 小寫字母
- 數字
- 特殊字元 (!@#$%^&*())

## 📊 視覺提示

出勤率色彩：
- 🟢 ≥90%: 優秀
- 🟠 80-89%: 良好
- 🔵 60-79%: 一般
- 🔴 <60%: 警告

- 側邊導航欄（可收合）

- 即時狀態回饋
- Material Design 風格
