# 實作現況說明

## 1. 系統型態
- 單機桌面版 (tkinter)
- SQLite 本機資料庫 (`data/handover_system.db`)
- 多語言 JSON 資源 (`frontend/public/locales/*.json`)

## 2. 已完成功能
- 登入流程與權限控管
- 日報表基本資訊門檻 (日期/班別/區域)
- 出勤記錄 (正職/契約同頁輸入)
- 設備異常 / 異常批次記錄
- Delay List / Summary Actual 匯入與上傳
- 出勤統計表 + 折線圖 + 堆疊柱狀圖
- 異常歷史查詢 (設備/批次)
- 管理員功能：使用者、翻譯、班別/區域
- 系統設定：備份間隔保存

## 3. 已知限制
- 語言偏好目前僅在單次使用期間保留
- 自動備份為設定值保存，實際備份需搭配外部排程或手動執行

## 4. 文件對齊
- 主要文件已更新至 v2.2
- 規格、計畫、設計、使用手冊已同步現況
