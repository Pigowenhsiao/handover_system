# 実装状況

## 1. システム形態
- 単機デスクトップ版 (Tkinter)
- SQLite ローカル DB（`data/handover_system.db`）
- 多言語 JSON リソース（`frontend/public/locales/*.json`）

## 2. 実装済み機能
- ログインと権限制御
- 日報基本情報ゲート（日付/シフト/エリア）
- 出勤記録（正社員/契約社員の同時入力）
- 設備異常 / 異常ロット記録
- Delay List / Summary Actual のインポートとアップロード
- 出勤統計表 + 折線図 + 積み上げ棒グラフ
- 異常履歴照会（設備/ロット）
- 管理者機能：ユーザー、翻訳、シフト/エリア
- システム設定：バックアップ間隔の保存
- テーマ切替：ライト/ダークと設定保存

## 3. 既知の制限
- 言語選択は同一セッション内のみ保持
- 自動バックアップは設定保存のみ。実際のバックアップは外部スケジュールまたは手動が必要

## 4. ドキュメント整合
- 主要ドキュメントは V 0.1.4 に更新済み
- 仕様/計画/設計/操作マニュアルは現況に同期済み
