# クイックスタートガイド (Quickstart Guide)

## 1. システム概要

本システムは単機デスクトップアプリ（Tkinter）です。日/中/英の切り替えに対応し、データは本機 SQLite（`data/handover_system.db`）に保存されます。ネットワークやサーバーは不要です。

## 2. インストールと起動

1. 依存関係のインストール：
   ```bash
   pip install -r requirements.txt
   ```
2. モダン UI の起動：
   ```bash
   python handover_system.py
   ```

既定の管理者アカウント：`admin` / `admin123`

## 3. 基本操作フロー

1. **ログイン** → 言語を選択 → アカウントでログイン
2. **日報** → 日付/シフト/エリアを選択 → 基本情報を保存
3. **他機能**（出勤/設備異常/ロット異常/統計/履歴照会等）は基本情報保存後に解除

> 同一 日付 + シフト + エリア は同一日報として扱い、再保存で上書きされます。

## 4. 主な機能

- 出勤：正社員/契約社員の定員・出勤・欠勤・理由
- 設備異常：設備番号、異常内容、発生時刻、影響数、対応内容、画像パス
- 異常ロット：ロット番号、異常内容、処置状況、特記事項
- 総結：出勤統計表 + 出勤率折線図 + 出勤人数の積み上げ棒グラフ
- 異常履歴：設備異常/ロット異常の履歴照会（期間 + シフト/エリア）
- Delay List / Summary Actual：Excel インポート → 仮保存 → アップロード
- テーマ切替：ライト/ダークを即時切替
