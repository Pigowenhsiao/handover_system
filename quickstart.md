# 快速入門指南 (Quickstart Guide)

## 1. 系統概述

電子交接本為單機桌面應用程式（Tkinter），支援中/英/日語切換，資料儲存於本機 SQLite（`data/handover_system.db`），不需網路或伺服器。

## 2. 安裝與啟動

1. 安裝依賴：
   ```bash
   pip install -r requirements.txt
   ```
2. 啟動現代化介面：
   ```bash
   python run_modern_system.py
   ```

預設管理員帳號：`admin` / `admin123`

## 3. 基本操作流程

1. **登入** → 選擇語言 → 輸入帳密登入
2. **日報表** → 選擇日期/班別/區域 → 儲存基本資訊
3. **其他功能**（出勤/設備異常/批次異常/總結/歷史查詢等）在基本資訊儲存後解鎖

> 同日期 + 班別 + 區域視為同一筆日報，重複儲存會覆寫舊資料。

## 4. 主要功能

- 出勤：正職/契約定員、出勤、欠勤與理由
- 設備異常：設備號碼、異常內容、發生時刻、影響數量、對應內容、圖片路徑
- 異常批次：批號、異常內容、處置狀況、特記事項
- 總結：出勤統計表 + 出勤率折線圖 + 出勤人數堆疊柱狀圖
- 異常歷史：設備異常/批次異常歷史查詢（日期區間 + 班別/區域篩選）
- Delay List / Summary Actual：Excel 匯入 → 暫存 → 上傳

## 5. 設定與備份

- 系統設定（備份間隔）保存在 `handover_settings.json`
- 建議定期備份 `data/handover_system.db`
