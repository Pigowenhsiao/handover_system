# 交接系統（桌面版） v0.3.4

Tkinter 製作的單機交接系統，支援中/英/日語切換，包含日報填寫、報表檢視與使用者/班別/區域管理，並提供 Delay List 與 Summary Actual 匯入/上傳。

## 主要功能
- 多語言介面：登入、主畫面、報表與管理頁可即時切換中/英/日。
- 日報填寫：輸入日期/班別/區域，紀錄出勤、設備異常、異常 LOT 與摘要；同日同班別同區域時會詢問是否覆寫。
- 報表：出勤/設備/LOT 報表可查詢、匯出 CSV；列表雙擊可開啟明細視窗。
- Delay List：從 `Z:\☆Junior Supervisor日報\Delay_List` 選取 Excel 匯入為暫存，確認後再「上傳」寫入 DB；可依日期範圍篩選並雙擊編輯（保留單一編輯視窗）。匯入時可選擇工作表。
- Summary Actual：讀取 `Summary(Actual)` 工作表，先匯入暫存、確認後再「上傳」入 DB；可依日期範圍篩選，數值 0 顯示為 `-` 並置中，雙擊可編輯（單一編輯視窗）。
- 管理：使用者/班別/區域列表支援雙擊編輯，並可新增、刪除、重設密碼。介面預設可捲動、可調整大小。

## 使用方式
1. 安裝依賴：`pip install -r requirements.txt`
2. 執行程式：`python app.py`
3. 登入前可先選擇語言；登入後在各分頁填寫日報、匯入/上傳報表或管理使用者與選項。

## 注意與限制
- Excel 匯入採「匯入 -> 暫存 -> 上傳」流程，未上傳不會寫入 SQLite。
- 同日期資料上傳會先刪除該日期既有記錄以避免重複。
- 系統為單機版（無前後端 API），使用 SQLite `handover_system.db`；如需重置，可備份或刪除該檔後重新啟動讓程式初始化。
