# 多言語対応 データモデル (Data Model)

多言語リソースは JSON ファイルで保存し、SQLite のテーブルは使用しません。

## 1. 言語リソースファイル
- 位置：`frontend/public/locales/*.json`
- 構造：ネストされたキー/値

例：
```json
{
  "header": {
    "title": "電子引き継ぎシステム",
    "login": "ログイン"
  },
  "common": {
    "save": "保存",
    "cancel": "キャンセル"
  }
}
```

## 2. 現在の言語設定
- LanguageManager が実行時に保持
- 既定言語は日本語 (`ja`)
- 再起動を跨ぐ永続化はしない

## 3. 検証ルール
- 言語コードは `ja`, `zh`, `en` のみ
- 翻訳キーはドット区切りで階層化
- キー欠落時は既定文言にフォールバック
